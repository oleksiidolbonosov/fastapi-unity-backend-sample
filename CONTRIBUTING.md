# Contributing

Thanks for your interest in contributing!

## How to contribute
1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Make changes and run tests: `pytest`
4. Commit your changes: `git commit -m "feat: short description"`
5. Push and open a Pull Request

## Code style
- Use `black` for Python formatting.
- Use meaningful commit messages (Conventional Commits recommended).

## Reporting issues
- Please open issues for bugs or feature requests using the issue template.

## Security
Do **not** commit secrets. Use environment variables and `.env` files.
